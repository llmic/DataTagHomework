# 工业互联网年报AI标注辅助工具 README
## 一、项目简介
本项目是**华南师大大一大二期中作业**专用自动化标注辅助工具，严格对齐作业要求，用于完成**上市公司年报工业互联网应用标注任务**。
- 任务目标：从12份上市公司年报中筛选关键词句子，标注企业是否**实际应用**工业互联网平台（已应用=1，未应用/规划中=0）
- 工具作用：自动分句、关键词筛选、本地大模型辅助标注、实时保存进度、一键生成符合提交格式的Excel文件
- 核心原则：100%遵循作业标注规则，无自定义修改，满足作业评分要求

## 二、环境准备（必须提前安装）
### 1. 基础软件
- Python 3.8及以上版本（官网下载安装，勾选Add Python to PATH）
- Ollama（本地大模型运行工具，必须安装）
  下载地址：https://ollama.com/

### 2. 拉取本地模型（安装Ollama后执行）
打开命令提示符（CMD），输入：
ollama pull deepseek-r1:8b

### 3. Python依赖库安装
打开CMD，进入项目文件夹，执行：
pip install regex pandas ollama

## 三、项目文件结构
项目根目录/
├── main.py                # 主程序文件（核心代码）
├── data_source/           # 【必须创建】存放12份年报txt文件
├── 单年报标注结果/        # 自动生成，单份年报标注结果
├── 临时标注进度.csv       # 自动生成，防止程序中断丢失数据
└── 班级+学号+姓名_总标注结果.xlsx  # 最终提交文件（自动生成）

## 四、运行步骤
1. 准备年报文件
   创建data_source文件夹，将分配的12份年报txt文件放入该文件夹
2. 检查配置
   无需修改代码，核心配置已100%对齐作业要求
3. 启动程序
   CMD进入项目目录，执行：
   python main.py
4. 自动标注
   程序自动完成：分句→关键词筛选→上下文提取→本地模型标注→实时保存进度
5. 生成结果
   标注完成后，自动生成：
   - 总提交Excel（班级+学号+姓名_总标注结果.xlsx）
   - 单份年报标注结果（存入「单年报标注结果」文件夹）
6. 提交文件
   重命名总Excel为：班级+学号+姓名.xlsx，交给班长汇总

## 五、核心功能
1. 合规关键词筛选
   严格使用作业117个关键词，长词优先匹配，无遗漏、无新增
2. 严格标注规则
   标注逻辑1:1照搬作业要求，区分「已应用/规划中/对外服务/模糊描述」
3. 进度自动保存
   每标注1句自动保存临时文件，断电/关闭程序不丢失数据
4. 格式自动对齐
   生成的Excel包含：id（股票代码）、year（年份）、句子原文、标注结果，完全匹配作业模板
5. 批量结果输出
   同时生成总表+单文件，满足作业每份≥15句、总计≥180句要求

## 六、注意事项
1. 关键词筛选合规
   程序按作业要求**人工逻辑筛选**（非AI批量筛选），符合作业“禁止AI批量筛选关键词”要求
2. 标注辅助合规
   仅用本地大模型辅助判断，**最终标注结果可人工核对**，符合作业规则
3. 禁止修改核心配置
   代码中「核心配置区」（关键词、标注规则、输出路径）严禁修改，否则不符合作业要求
4. 文件名规范
   年报文件名必须为：证券代码_年份_公司简称_年度报告类型_日期.txt，否则无法解析股票代码/年份
5. 模型必须本地运行
   禁止使用在线AI批量标注，仅允许Ollama本地模型辅助(调整之后可在线，毕竟qwq大厂的肯定比咱的好)
6. 人工复核
   标注完成后建议简单核对，确保无明显错误（如规划句标1、应用句标0）

## 七、作业提交要求
    文件命名：班级+学号+姓名.xlsx

## 八、异常处理
- 年报无法打开/乱码：联系学生助理更换备选年报
- 模型调用失败：检查Ollama是否启动、模型是否拉取成功
- 程序报错：确认依赖库安装完整、年报文件格式正确
